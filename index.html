To create a button:

  1. Add the cta class to your link.
  2. Add a second class to choose the color: cta-red, cta-green, cta-blue, or cta-yellow.
  3. Add an icon using classes: icon- plus:
    • check
    • doc
    • download
    • email
    • go
    • more
    • search

    4. I used an icon font from Fontello.
