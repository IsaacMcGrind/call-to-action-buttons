Designing the Buttons

You have two choices when creating call-to-action buttons: use CSS or an image. CSS allows you to quickly change the button; great for A/B testing. An image gives you greater design freedom and will render the same across all browsers. I usually opt for an image as I've done on the Pixify homepage, but I'm going to use CSS today so you can easily customize these buttons.

To create a button, simply add the cta class to your link. Then, add a second class to choose the color: cta-red, cta-green, cta-blue, or cta-yellow. To include an icon, add a span tag iside the link and give it one of these classes: icon-download, icon-check, icon-go, or icon-more. I used icons from tutorial9's 108 Mono Icons pack.
